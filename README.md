# face_recognition

### 1. 간단한 얼굴인식, 코와 입 검출을 하는 파이썬 프로그램  

### 2. 웹캠에 접속하여 사진(ID)을 등록하고 학습하여 해당 ID에 맞는 얼굴을 검출하는 파이썬 프로그램


## 사용

Python & OpenCV
